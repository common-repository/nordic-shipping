Create and track and manage shipments, print shipping labels from your dashboard. And you can select pick-up points also from your dashboard.

== Description ==

**This connector plugin is for users utilizing Shipit for managing their shipment, and want to save time on creating the shipping labels. You can sync the order details to Shipit with a single click and print shipping labels from there. You can also print the labels directly from within the plugin. The app support all major carriers in Sweden and Finland such as Postnord, Posti, UPS, Matkahuolto, FedEx, Bring, Schenker, GLS etc.**

For fetching shipping methods and pickup points we are using shipit api.

shipit test api : https://apitest.shipit.ax/v1/

shipit live api : https://api.shipit.ax/v1/

shipit api documentation : https://shipit.fi/docs/api-docs/#/

shipit privacy policy : https://www.shipit.fi/rekisteriseloste/

**Key features**

1. Automatic sync
2. Manual sync
3. Print shipping labels
4. Support carrier agents
5. Track shipments
6. Tracking link in e-mails

**How to use**

1. Activate the plugin
2. Then Click on " Nordic shiping " menu. Then update the settings like sender info etc.
3. You have the option to sync the orders from the WooCommerce order detail page to the Shipit dashboard once you've configured the necessary settings on the Nordic Shipping Settings page.

== Installation ==

1. Upload "nordic-shipping.zip" and install.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Nordic Shipping menu and click to start onboarding
4. Go to WooCommerce settings and setup shipping as usually, choose Nordic Shipping as method
1. Add description, price, shipping methods, service point (if supported by method), prices and shipping rules.
6. Save
